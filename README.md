## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/minimal-portfolio.git
   cd minimal-portfolio
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Tech Stack
- React 
- Vite 
- Tailwind CSS
